Galileo
-------

1) Unzip Crypto.zip to a folder named 'Crypto' and place in /usr/lib/python2.7/site_packages

2) open nrf_crypto.ino and nrf_crypto.h in the Arduino IDE

3) build and program the Galileo w/ nrf_crypto

4) run shm_decrypt.py on the Galileo


MSP430
------

1) run the build command in build.txt

2) upload to msp430
